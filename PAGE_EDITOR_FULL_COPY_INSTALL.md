# ASORTA v6.4.3 — Full Page Editor hotfix

Deze hotfix herstelt `/atlas/pages` en maakt de vaste klantgerichte pagina-copy beheerbaar vanuit Atlas Page Editor.

Belangrijk:
- Producttitels, prijzen, productbeschrijvingen en productmedia blijven via Shopify/Product Editor lopen.
- Deze editor beheert vaste pagina-copy: homepage, shop, about, contact, FAQ, shipping, returns, privacy, terms, checkout, success, account, track order, footer en globale snippets.
- Run `supabase/v5_23_full_page_editor_schema.sql` in Supabase SQL Editor.
