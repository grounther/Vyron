# VYRON Store v3

Premium Next.js + Tailwind ecommerce starter.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub. Vercel should use:
- Framework: Next.js
- Build command: npm run build
- Install command: npm install
- Output directory: empty
